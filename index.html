Character Sprites are used in game design

A group of images can be made into spritesheets which can be used to animate a video game character to move, jump, climb, the limits are endless!

Levels can be made from sprites!

Learning how to make levels in a doodling software is just a matter of getting started. It takes a long time to create levels but a way to start is make a sprite sheet of objects like doors to start.
